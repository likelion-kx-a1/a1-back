# Claude 백엔드 소스코드 자동 생성용 마스터 프롬프트

> AI 에이전트에게 a1-back의 Spring Boot 소스코드를 처음부터 생성시킬 때 사용했던 프롬프트 원본입니다. 실제 코드/문서를 만들기 위한 도구이며, `docs_h/`의 실무 기술 문서와는 별개로 이 폴더(`docs_p/`)에 격리 보관합니다.

[역할 설정]
너는 10년 이상의 실무 경력을 가진 수석 Java Spring Boot 백엔드 아키텍트이자 개발자이다.
제공된 설계 사양서와 비용 극소화 테스트 가이드를 바탕으로, 즉시 컴파일 및 프로덕션 배포가 가능한 수준의 완벽한 Java 소스코드를 작성하라.

[참고 사양 문서]

- 백엔드 API 연동 설계서
  - Claude-Sonnet, ChatGPT Image, fal.ai API의 동기/비동기 오케스트레이션 파이프라인.
  - Spring Boot DDD + 실용적 계층형 하이브리드 아키텍처 적용.
  - 5초 간격의 Spring `@Scheduled` 기반 백그라운드 비디오 완성 상태 추적 및 AWS S3 업로드 처리.
  - PostgreSQL 기반 `image_assets`, `video_tasks` 테이블 구조 연동.
- 비용 극소화 테스트 가이드
  - 로컬 환경(`local` 프로필)에서 fal.ai API 비용을 단 1원도 쓰지 않고 시뮬레이션할 수 있는 Mock 구현체 구조 탑재.

[요구사항 및 구현 규칙]

1. **No Shortcuts (코드 생략 절대 금지)**: 클래스 작성 시 중간에 `// 생략...` 또는 `// TODO: 로직 구현` 등의 주석 처리를 하지 말고, 필드 선언부터 import문, 예외 처리, 비즈니스 로직 끝까지 실제 동작하는 코드로 완전하게 작성하라.
2. **Profile 기반 다중 구현체**: 외부 API 연동 인터페이스(`PromptEnhancer`, `ImageGenerator`, `VideoGenerator`)를 설계하고, 개발/로컬용 가짜 구현체(`Mock...` - `@Profile("local")`)와 운영용 실제 구현체(`... - @Profile("prod")`)를 완벽하게 분리하라.
3. **AWS S3 및 가짜 파일 처리**: 실제 프로드 환경에서는 외부 API가 반환한 임시 URL로부터 Stream을 열어 S3 버킷으로 직접 멀티파트 업로드를 처리하는 코드를 구현하고, 로컬 환경에서는 미리 준비된 안전한 더미 미디어 URL을 반환하도록 설계하라.
4. **Spring Boot 3.x 관례 준수**: Lombok 어노테이션(`@RequiredArgsConstructor`, `@Getter` 등), Spring Data JPA, RestTemplate 또는 WebClient를 활용한 깔끔한 HTTP 요청 코드를 작성하라.

[출력해야 할 소스코드 목록]

아래 나열된 모든 자바 클래스를 차례대로, 완성된 형태의 코드로 출력하라.

**Domain Layer (Entities & Enums)**
- `com.likelion.a1.generation.domain.ImageAsset` (JPA Entity)
- `com.likelion.a1.media.domain.VideoStatus` (Enum)
- `com.likelion.a1.media.domain.VideoTask` (JPA Entity)

**Infrastructure Layer - 추상화 인터페이스 및 다중 구현체**
- `com.likelion.a1.generation.application.PromptEnhancer` (인터페이스)
- `com.likelion.a1.generation.infrastructure.ClaudePromptEnhancer` (Claude API 호출)
- `com.likelion.a1.generation.infrastructure.MockPromptEnhancer` (가짜 한영 보정 엔진)
- `com.likelion.a1.generation.application.ImageGenerator` (인터페이스)
- `com.likelion.a1.generation.infrastructure.OpenAiImageGenerator` (DALL-E 이미지 생성)
- `com.likelion.a1.generation.infrastructure.MockImageGenerator` (가짜 이미지 에뮬레이터)
- `com.likelion.a1.media.application.VideoGenerator` (인터페이스)
- `com.likelion.a1.media.infrastructure.FalVideoGenerator` (fal.ai 비동기 제출 및 상태 조회)
- `com.likelion.a1.media.infrastructure.MockVideoGenerator` (비용 $0 보장 비디오 에뮬레이터)

**Storage Service**
- `com.likelion.a1.global.config.S3Config` (S3 클라이언트 빈 설정)
- `com.likelion.a1.global.infrastructure.S3UploadService` (임시 URL 파일을 다운로드하여 S3 버킷으로 멀티파트 업로드 처리하는 최적화 코드)

**Application Logic & Scheduler**
- `com.likelion.a1.generation.application.GenerationApplicationService` (사용자 입력 수신 → 1단계 Claude 프롬프트 보정 → 2단계 ChatGPT 이미지 생성 및 S3 저장 → 3단계 fal.ai 비동기 비디오 큐 제출 및 Task DB 생성의 흐름을 조율하는 서비스 클래스)
- `com.likelion.a1.media.application.VideoScheduler` (5초 주기로 DB에서 PROCESSING 상태인 비디오 태스크들을 조회해 fal.ai 상태를 체크하고, 완료 시 S3 업로드 및 DB 업데이트를 자동 처리하는 배치 스케줄러)

**Presentation Layer (Web Controller)**
- `com.likelion.a1.generation.presentation.GenerationController` (`POST /api/generate`, `GET /api/video-status/{taskId}` API 엔드포인트 구현)

준비가 완료되었다면, 위 패키지 경로에 맞추어 생략 없는 순수 자바 코드를 순서대로 출력하라.
